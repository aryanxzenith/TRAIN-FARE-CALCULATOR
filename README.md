# TRAIN-FARE-CALCULATOR
🚆 Train Fare Calculator (C++)

📌 Description
This project calculates train fare based on travel distance, selected class, and shortest route between stations using Data Structures and Algorithms.

🚀 Features
✔ Multiple train classes (General, Sleeper, AC)  
✔ Distance-based fare calculation  
✔ Shortest path between stations  
✔ Fast route searching using DSA  
✔ User-friendly menu system  

🧠 DSA Concepts Used
✔ Graph (Adjacency List)  
✔ Hash Map (`unordered_map`)  
✔ Priority Queue  
✔ :contentReference[oaicite:0]{index=0}  

🧮 Logic
Shortest Distance = Calculated using Dijkstra Algorith
Fare = Distance × Rate per km  

General = ₹2/km  
Sleeper = ₹3/km  
AC = ₹5/km  

💻 How to Run
Compile: g++ main.cpp -o train  
Run: ./train  

📷 Sample Output
Enter source station: Delhi  
Enter destination station: Lucknow  
Select Class: 2  

Shortest Distance: 530 km  
Total Fare: ₹1590
